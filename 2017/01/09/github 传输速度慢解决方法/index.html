github 传输速度慢解决方法

Posted on 一月 9, 2017

github被防火长城限制了？？怎么办？

原来是这个域名github.global.ssl.fastly.NET被限制。很简单，去查查这个域名对应的ip，然后修改自己电脑的hosts文件

查询地址：http://tool.chinaz.com/dns

选择一个最小的TTL值，最小一般为30

然后打开C->windows->system32->driver->etc->hosts,在最后一行加入如下一句：

151.101.72.249 github.global.ssl.fastly.Net

这样本来5-6k的传输速度提高到600k左右了！！
